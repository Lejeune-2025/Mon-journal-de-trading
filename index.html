Installer l'application

Installez le journal sur votre écran d'accueil pour un accès rapide, comme une application native.

Lancer en mode application

Pour installer la PWA et le mode offline complet, lancez le serveur local inclus :

  1. Windows : double-cliquez start.bat dans le dossier du projet
  2. Ou en terminal : npx serve . -p 3000
  3. Ouvrez http://localhost:3000 dans Chrome ou Edge

Synchroniser avec votre téléphone

Les données sont stockées localement par appareil. Pour les retrouver sur mobile :

  1. Copiez le dossier journal-trading sur votre téléphone (USB, Google Drive, WhatsApp…)
  2. Ouvrez index.html dans Chrome/Safari mobile
  3. Ou exportez vos données ci-dessous et importez-les sur le téléphone

L'export inclut tous vos trades, captures d'écran et notes.
